🎨 Customization

Color Scheme

The template uses CSS custom properties for easy theming:

```css
:root {
    --primary: #4e54c8;
    --secondary: #8f94fb;
    --accent: #ff6b6b;
    --dark: #2c2c54;
    --light: #f7f7f7;
}
```

Dark Mode

Toggle between light and dark themes using the switch in the header or respect system preferences.
